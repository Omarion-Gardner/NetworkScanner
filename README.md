# NetGuard — Network Log Scanner

A network log analysis tool that scans simulated network traffic logs for threats using signature-based detection, IP tracking, and prioritized alert generation.

## Features
- Log Parsing & Packet Simulation
- Signature-Based Detection
- Alert Generation with Severity Scoring
- IP Tracking & Port Scan Detection
- Event Logging & Audit Trail

## Data Structures Used
- ArrayList, HashMap, LinkedList, Stack, PriorityQueue

## How to Run
1. Clone the repository
2. Open in VS Code
3. Run `Main.java`
